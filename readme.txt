== Description ==

This Plugin is a very simple Plugin and allows you to add with the shortcode [cn] the Copyright notice and auto changes the year display. 

== Frequently Asked Questions ==

=How can i use it?=

Just activate the plugin and put the shortcode [cn] where you want the text to display.
